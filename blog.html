Important FAQ ?

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

2. How do you select an element from the DOM?

I can select an element by using Tag Name, by ID, by Class Name, by Query Selector and Query Selector All.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows us to manage events more efficiently, especially when dealing with multiple elements. Instead of attaching event listeners to each child element individually, we add a single event listener to a common parent element.

4. How do you manipulate an element's attributes and styles using the DOM?

We can use methods such as getAttribute, setAttribute, removeAttribute to work with attributes.
To manipulate an element's style attribute using the DOM, we can use the style property, which allows us to set or modify individual CSS properties for that element.
